win10下安装face_recogniton

发表于 2018-04-17   |   分类于 人脸识别

今天给大家介绍一下一个非常厉害的一个人脸识别库face_recognition.
这个项目的GitHub地址：https://github.com/ageitgey/face_recognition
这是一个python的库，所以首先我们需要安装一下python.python的官网地址：https://www.python.org/
安装完python（随便记得把python加到路径中）。
接下来这一步比较麻烦了，由于face_recognition需要dlib库，这个库安装比较麻烦。
第一.我们需要装vs，推荐大家装vs2015.
第二.安装cmake，官网下载安装包：https://cmake.org/download/
第三.安装boost（这个需要记一下安装地址），下载地址：http://www.boost.org/
都下载好之后我们就可以动手来安装dlib了。
首先找到vs2015的本机工具命令提示符工具

VS2015开发人员命令提示，可以在这里编Boost,是一个CMD窗口.cd 进入进入之前下载的Boost目录，执行bootstrap.bat
再在命令中输入b2 install
然后我们就可以利用b2编译库文件
在cmd中输入：
b2 -a –with-python address-model=64 toolset=msvc runtime-link=static
之前你cmake下载的64位address-model这里写64，如果是32位的就把之前的64改成32
然后设置变量
　　 set BOOST_ROOT=C:\local\boost_1_63_0
　　 set BOOST_LIBRARYDIR=C:\local\boost_1_63_0\stage\lib
这里变量对应着你boost下载的目录
接下来我们就可以安装dlib了
下载dlib，解压，然后运行在dilb的目录下运行
python setup.py install
最后我们就可以安装face_recognition了，我们只要在cmd中运行
pip install face_recognition
pip就会自动帮你把其他face_recognition的依赖库都下载安装好。

之后我们就可以在python中愉快的使用这个人脸识别库了！

至于用法撒的，在项目的github上讲解的非常清楚了，我就不多啰嗦了。
下一次我就来介绍一下如何利用这个库做一下人脸评分。
