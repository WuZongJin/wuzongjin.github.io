算法之数的排序

发表于 2017-11-29   |   分类于 算法

数字的排序！

在编写程序时常常需要对一些数据按大小进行排序，我今天就来介绍以下排序的3种算法：1.桶排序，2.冒泡排序，3.快速排序。

1.桶排序

这是一个非常快速的排序方法，其时间复杂程度只有O(m+n)。其主要思想就是申请一个足够大的空间，就是一个个桶一样，
然后对于所要排序的数字进行读取，并且在相应的‘桶’中数字加一。
下面给一个例子：
//对于10以内的数字进行排序。

#include<stdio.h>
int main()
{
    int a[11] = { 0 };
    int t;
    printf("请输入需要排序的数字：");
    for (int i = 0; i<10; i++)
    {
        scanf("%d", &t);
        a[t]++;
    }


    for (int i = 0; i<11; i++)
        for (int j = 1; j <= a[i]; j++)
            printf("%d  ", i);

    return 0;
}

该算法虽然快，但是非常浪费空间。如果要对0~1000000000之间的数进行排序，那么就要申请1000000001个变量，也就是说即使你要对1，2009，9898，756321，9999999这5个数字进行排序时也得要申请那么多空间，而且对于小数的排列，该算法也不能很好的解决，所以这就需要另外的排序方法。

2.冒泡排序

冒泡排序应该是我们学生最经常用的一种排序方法了。其主要思想：每次比较两个相邻的元素，如果他们的顺序错误就交换它们。
例子：

#include<stdio.h>
int main()
{
    int n;
    printf("要对多少个数进行排序：");
    scanf("%d", &n);
    int *a = new int [n];
    for (int i = 0; i<n; i++)
    {
        scanf("%d", &a[i]);
    }
    for (int i = 0; i<n - 1; i++)
        for (int j = 0; j<n - 1; j++)
        {
            if (a[j]>a[j + 1])
            {
                int t = a[j];
                a[j] = a[j + 1];
                a[j + 1] = t;
            }
        }

    for (int i = 0; i<n; i++)
        printf(" %d", a[i]);
    return 0;
}

冒泡排序解决了桶排序浪费空间的问题，但是在算法的执行效率上却牺牲了很多，它的时间复杂程度达到了O(N²)。

3.快速排序

上面介绍的两种算法都有其优缺点，那么有没有一种既不浪费空间又可以快一点的排序呢？那就是接下来我要介绍的“快速排序”啦！
这也是我在这里重点介绍的！
假设我们现在对“6 1 2 7 9 3 4 5 10 8”这10个数字进行排序。首先要做的是在这个序列中找一个数字作为基准数。
为了方便，我们就让第一个数6作为基准数。接下来将这个序列中所有比基准数大的数放在右边，比基准数笑的数
放在左边。于是就得到：
3 1 2 5 4 6 9 7 10 8
初始状态下6为于第一位，现在我们将6挪到了中间的某个位置，然后将这一数列分为2份，接下来再对6的左边和右边再进行相同的操作。得到的结果如下：
2 1 3 5 4
8 7 9 10
接下来继续，将数列分为3的左边和右边，以及9的左边和右边。我们发现9的右边只有一位10.那就说明基准数10已经归位。因此只要将这个步骤一直执行到基准数都归为为止，那么排序就完成了。

其实现的代码如下：

#include<stdio.h>
int a[101], n;//定义全局变量

void quicksort(int left, int right)
{
    int i, j, t, temp;
    if (left > right)
        return;

    temp = a[left];//temp就是基准数
    i = left;
    j = right;
    while (i != j)
    {
        //先从右往左找
        while (a[j] >= temp&&i < j)
            j--;
        //再从左往右找
        while (a[i] <= temp&&i < j)
            i++;
        //交换2个数在数组中的位置
        if (i < j)//当i和j还没有相遇时
        {
            t = a[i];
            a[i] = a[j];
            a[j] = t;
        }
    }
    //最后再将基准数归位
    a[left] = a[i];
    a[i] = temp;

    quicksort(left, i - 1);//继续处理左边的
    quicksort(i + 1, right);//继续处理右边的
    return;
}

int main()
{
    int i, j;
    //读入数据
    printf("要排序的数字有多少个：");
    scanf("%d", &n);
    printf("要排序的数字为：");
    for (i = 1; i <= n; i++)
    {
        scanf("%d", &a[i]);
    }

    quicksort(1, n);//快速排序调用

    //输出结果
    for (i = 1; i <= n; i++)
    {
        printf("%d ", a[i]);
    }
    return 0;
}
